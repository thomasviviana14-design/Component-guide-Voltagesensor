Voltage Sensor Module

Voltage Sensor Module (0-25V) for Microcontrollers

Introduction

A **Voltage Sensor** is a simple but vital module used to measure the amount of electricity flowing through a circuit. Since microcontrollers like the ESP32 or Arduino cannot safely measure high voltages (anything above 3.3V or 5V) directly, this sensor acts as a "translator." It scales down a large voltage so your controller can read it safely without blowing up.

How it Works

This module is based on a **Voltage Divider** principle. It uses two precisely chosen resistors to reduce the input voltage by a factor of 5. For example, if you connect a 25V battery to the input, the sensor "divides" it down to 5V at the output. Your code then multiplies that reading by 5 to tell you the true voltage of the battery.

How it is Used

It is most commonly used for battery management and monitoring systems. Key uses include:

  • Monitoring the charge level of a Li-ion or Lead-Acid battery.
  • Checking the output of a Solar Panel in real-time.
  • Creating a digital DIY Multimeter using an OLED screen.
  • Protecting circuits by triggering an alarm if the voltage gets too high or too low.

Wiring: Note the distinction between Input (Screw Terminals) and Output (Pins)

Understanding the Pins

The voltage sensor has two sides: a High Voltage input side and a Low Voltage output side.

  • VCC (Input): The positive terminal of the voltage you want to measure.
  • GND (Input): The negative terminal of the voltage source you are measuring.
  • S (Signal): The output pin that sends the scaled-down voltage to your microcontroller's Analog (ADC) pin.
  • + (Output): This pin is usually not connected for basic use.
  • - (Output): Connect this to the Ground (GND) of your microcontroller.

Technical Summary

This sensor is rated for a range of **0V to 25V DC**. It is important to remember that the maximum output depends on your microcontroller. If you are using an Arduino (5V), you can measure up to 25V. However, if you are using an **ESP32 (3.3V)**, you should only measure up to **16.5V** to ensure the "Signal" pin never exceeds the 3.3V limit of your board!
